# SQL Queries for Danny's Restaurant Data Analysis

## Overview
This folder contains SQL queries used to analyze customer transactions at a restaurant.  
The queries answer important business questions about sales, customer behavior, and loyalty program points.

## Files in this Folder
- `01_total_spent.sql` → Total amount each customer spent at the restaurant
- `02_days_visited.sql` → Number of days each customer visited the restaurant
- `03_first_item.sql` → First item purchased by each customer
- `04_most_purchased_item.sql` → Most purchased item overall
- `05_most_popular_item_per_customer.sql` → Most popular item per customer
- `06_first_item_after_membership.sql` → First item purchased after joining loyalty program
- `07_last_item_before_membership.sql` → Item purchased just before joining membership
- `08_total_items_before_membership.sql` → Total items and amount spent before membership
- `09_points_calculation.sql` → Points calculation (sushi 2x)
- `10_points_first_week.sql` → Points calculation including first week 2x points for all items
